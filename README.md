# elastic-fruit-runner

Elastic GitHub Actions self-hosted runner manager for Apple Silicon.

- **macOS runner** via [Tart](https://tart.run) — ephemeral VMs, auto-scaled
- **Linux arm64 runner** via Docker / OrbStack — native performance
- **Linux amd64 runner** via Docker + Rosetta 2
- Powered by GitHub Runner Scale Set Client
