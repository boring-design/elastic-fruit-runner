## Features

<CardGrid stagger>
	<Card title="macOS Runners (Tart)" icon="apple">
		Ephemeral macOS VMs on Apple Silicon via [Tart](https://tart.run). One VM per job, auto-destroyed after completion.
	</Card>
	<Card title="Linux Runners (Docker)" icon="linux">
		Linux arm64 and amd64 runners via Docker-in-Docker. amd64 supported through Rosetta 2 emulation.
	</Card>
	<Card title="Host Mode" icon="rocket">
		Run the GitHub Actions runner directly on the machine with zero VM overhead for maximum performance.
	</Card>
	<Card title="Official Scale Set API" icon="github">
		Powered by the official [GitHub Runner Scale Set Client](https://github.com/actions/scaleset) for reliable job polling and runner lifecycle management.
	</Card>
</CardGrid>
